## Variant: Command Center

### Design stance
A dense power-user screen for scanning many candidates quickly while keeping a detail panel open.

### Key choices
- Layout: top stat strip, dense table, right-side detail and bucket panels.
- Typography: compact sans-serif with tabular financial data.
- Color: blue focus with risk/capture chips.
- Interaction: bucket filters update the table and clicking a row updates detail panel.

### Trade-offs
- Strong at: high-volume scanning and fast filtering.
- Weak at: risks repeating the old table/dashboard problem.

### Best for
Secondary “원자료/스캔” screen, not the default product home.
